## How to Use

1. Open `index.html` in your browser
2. Browse students in the grid
3. Use search to find specific students
4. Click any student card to view their full profile
5. Profiles are accessible via `/profile/{id}` in the URL
6. Click the theme toggle (bottom-right) to switch dark/light mode
